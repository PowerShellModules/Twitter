# Get-Tweet

## SYNOPSIS
Returns a single Tweet, specified by the TweetId parameter. The Tweet's author will also be embedded within the Tweet.

## SYNTAX

```powershell
Get-Tweet [-TweetId] <Int64> [<CommonParameters>]
```

## DESCRIPTION
Returns a single Tweet, specified by the TweetId parameter. The Tweet's author will also be embedded within the Tweet.
