## Building from the GitHub Repo

1. Clone repo to a folder of your choice.
2. Change into the local repo folder.
3. Run the `build.ps1` script.
4. `Import-Module -Path .\BuildOutput\BluebirdPS\0.1.0\BluebirdPS.psd1`
